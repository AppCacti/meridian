## Structure

- `index.html` — static one-page website.
- `styles.css` — standalone visual system.
- `script.js` — tiny header behavior.
- `assets/meridian-private-deal-desk-hero.png` — generated project-bound hero/social image.
- `public/` — deploy this folder publicly. It contains only the website and assets, not internal strategy docs.
- `docs/` — launch, sales, legal, and delivery documents.
- `START_HERE_RUNBOOK.md` — exact order of operations to get site live, push outreach, book calls, close first cash, and scale.
- `DEPLOYMENT.md` — public launch checklist for Cloudflare Pages, Netlify, or Vercel.
